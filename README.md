## üöÄ Getting Started

### Prerequisites
- Node.js & npm installed.
- A valid Google Gemini API Key.

### Development Setup
1. Clone the repository.
2. Ensure your environment variable `API_KEY` is configured.
3. Run the application in your browser.

*Note: To bundle this as a Chrome Extension, follow the [Chrome Extension Developer Guide](https://developer.chrome.com/docs/extensions/mv3/getstarted/) to register the build directory.*
